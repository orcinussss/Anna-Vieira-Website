# Project Documentation

## Overview
This project is the Anna Vieira Website, a personal website showcasing the portfolio and services of Anna Vieira.

## Setup Instructions

1. **Clone the Repository**  
   To clone the repository, run:
   ```bash
   git clone https://github.com/orcinussss/Anna-Vieira-Website.git
   ```

2. **Install Dependencies**  
   Navigate into the project directory and install dependencies:
   ```bash
   cd Anna-Vieira-Website
   npm install
   ```

3. **Run the Project**  
   Start the local development server:
   ```bash
   npm start
   ```

4. **Build for Production**  
   To create a production build, run:
   ```bash
   npm run build
   ```
